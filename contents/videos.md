# Application

## NBIB Literature Review Management Assistant

This is a NBIB Literature Review Management Assistant developed using React, capable of parsing PubMed-exported NBIB format files. It provides a visual literature library browser, intelligent categorization and chapter assignment functions, and automatically generates academic-standard Chinese review writing instructions (Prompts). It helps researchers efficiently organize literature, build review frameworks, and assist in AI writing, with all data stored locally for security.

## Clinical Data Analysis System

This is a zero-code interactive data analysis platform specifically designed for clinical researchers. The system integrates five core modules: data overview, Table1 generation, logistic regression, restricted cubic spline (RCS) analysis, and subgroup analysis, supporting multiple data formats including CSV, Excel, Stata, SPSS, and SAS. Through drag-and-drop operations and SCI-style visualizations, users can complete the entire workflow from data cleaning to advanced statistical modeling without any programming, generating publication-ready results that significantly enhance research efficiency and analytical standardization.

## Predictive Modeling Analysis System

This is a comprehensive predictive modeling analysis system developed using R Shiny, integrating LASSO variable selection, logistic regression modeling, and multiple machine learning algorithms (GBM/SVM/Random Forest). It provides a complete visual solution from data import, feature selection, and model construction to three-in-one performance evaluation (ROC curves, calibration curves, decision curve analysis). Supporting multi-format data import and professional chart export, it enables zero-code development and validation of clinical prediction models.
